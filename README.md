# Employee Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on an employee dataset using Python.

## Objectives
- Understand the employee dataset
- Clean missing and duplicate data
- Perform exploratory data analysis
- Visualize employee trends
- Generate useful insights

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Files
- Employee_Data_Analysis.ipynb
- employee_data.csv

## Analysis Performed
- Data Cleaning
- Missing Value Treatment
- Duplicate Removal
- Outlier Detection
- Univariate Analysis
- Bivariate Analysis
- Multivariate Analysis

## Key Insights
- Identified salary distribution across employees.
- Compared departments using visualizations.
- Analyzed relationships between important variables.
- Generated business recommendations based on the analysis.
